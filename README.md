# cordova-plugin-autotime

> Get auto time is enabled in device


## Install

```
$ cordova plugin add --save https://github.com/matoooon/cordova-plugin-autotime.git
```


## Usage

```js
window.plugins.getAutoTime(function callbackSuccess(res){
  //res true or false
})

```
